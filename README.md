# Садиба "Чугайстер"

## Опис проєкту
Цей проєкт є навчальним вебсайтом, створеним у межах дисципліни "Front-end розробка у Web".

Сайт представляє собою веб-сторінку садиби "Чугайстер", яка надає послуги відпочинку в Карпатах.

## Функціонал сайту
- Головна сторінка з інформацією про садибу
- Сторінка з варіантами проживання (будиночки)
- Сторінка бронювання
- Форма бронювання з валідацією
- Динамічні елементи (таби, FAQ)
- Адаптивний дизайн

## Технології
- HTML5 (семантична структура)
- CSS3 (Flexbox, адаптивність, стилізація)
- JavaScript (валідація, взаємодія з користувачем)
- jQuery (таби, анімації, інтерактивні елементи)
